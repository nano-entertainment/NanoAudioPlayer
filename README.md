NanoAudioPlayer
===============

Javascript
===============
```js
$('.audio-player').NanoAudioPlayer({ });
```
settings:
```js
{
  file: 'sound/sound',
  autoplay: true
}
```

Html
===============
```html
<div class="audio-player">
  <div><a class="play" href="#">Play</a></div>
  <div><a class="pause" href="#">Pause</a></div>
  <div><a class="play-toggle" href="#">PlayToggle</a></div>
</div>
```
